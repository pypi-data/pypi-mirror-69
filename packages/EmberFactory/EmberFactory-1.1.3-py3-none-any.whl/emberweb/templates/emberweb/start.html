All you have to do is to upload an Excel sheet containing the data in an appropriate format. Two formats are available:

  • Basic format example The basic format is identical to the presentation provided in IPCC's SRCCL supplementary material tables.
  • Fullflex format example The "full flexibility" format provides more flexibilty and control of the result (the first sheet from this Excel workbook provides more control on the risk levels, e.g. a transition that has a faster increase in risk at some temperature level; the second sheet provides graphic parameters to customize the presentation of your data, and can be copied in the Basic format so as to get an intermediate format, with simple presentation of data + more graphic parameters).

You can use those examples as provided to test the application, and you can also use them as templates to provide your own data. For a first test, we suggest that you simply download, upload, and run the Basic format example. To make embers from your own data, start from the Basic format and copy a standard IPCC-style table (see above). Documentation will be produced in the future, but we think that at least the Basic format can be understood without it in many cases.

Feedback is welcome, including on what you would like to have as input format.
Combinations of the Basic and Fullflex formats are possible (start from the Basic format and add the parameter and/or color sheet(s) from the flex format).
