Remember the following constraints with this tool:

  • A limited number of address group nesting is done. Configure this using the 'nesting' setting
  • The search does not include subnets which may contain the object
  • Rules with 'any' is omitted
  • Objects with a different name, but the same IP address is not searched for
  • Dynamic groups based on tags are not supported

This list is not a full list of all limitations. If you think this search should have resulted in hits, open an issue in the Github repository.
