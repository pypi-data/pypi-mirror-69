*.gt files
==========
Tests 1-6 are from Shanbhag, MTS 2019 paper. Test 1-3 are validation tests.

test1: Bimodal, Linearly Spaced with Noise
test2: Multimodal, Logarithmically Spaced
test3: Twin Peaked Spectrum
test4: Symmetric Star Theory
test5: Slip Link Simulation of Binary Linear Blend
test6: Experimental data on Ring Polymers
test7: test3 with artificially added G0 = 10.0

*.tts files
===========
Tests 1-3 are from Takeh et al. paper.

test1.dat : synthetic twin-peaked spectrum
test1n.dat: same twin-peaked spectrum + noise
test2.dat : symmetric star data
test3.dat : Watanabe binary LLB
test4.dat : Gel data with plateau-presubtracted
test4f.dat: Same gel data with plateau left intact (need to set plateau flag to on)
test5.dat : Hamed Janani EPDM rubber data
test6.dat : H3A1 comb data from Dealy
test7.dat : Laponite superposed G*(w)
test8.dat : example that violates Kramers-Kroenig
